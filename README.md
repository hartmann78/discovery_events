Discovery events
---
Discovery events представляет собой приложение, которое позволяет пользователям делиться информацией об интересных событиях и находить компанию для участия в них.
Проект создан на языке Java 17, с использованием фреймворка Spring Framework и собран на Apache Maven.

## Структура проекта
Данный проект состоит из 2-х сервисов:
- Сервис статистики: собирает информацию о количестве обращений пользователей к спискам событий и о количестве запросов к подробной информации о событии.
- Основной сервис: разделен на 3 части
    - Публичный API:
         - Категории;
         - События;
         - Подборки;
    - API для авторизованных пользователей:
        - События;
        - Запросы;
        - Комментарии;
    - API для администраторов;
        - Пользователи;
        - Категории;
        - События;
        - Запросы;
        - Подборки;
        - Комментарии.

## Почему проект полезен?
Свободное время — ценный ресурс. Ежедневно мы планируем, как его потратить — куда и с кем сходить. Сложнее всего в таком планировании поиск информации и переговоры. 
Нужно учесть много деталей: какие намечаются мероприятия, свободны ли в этот момент друзья, как всех пригласить и где собраться.

Приложение представляет собой афишу, в которой можно предложить какое-либо событие от выставки до похода в кино и собрать компанию для участия в нём.

## Как запустить проект?
### 1. База данных
1. Скачать и установить PostgreSQL при его отсутствии;
2. Открыть pgAdmin 4 и создать базы данных с названиями "events-service" и "stats-service";
### 2. Проект
1. Загрузить проект из ветки "main" по ссылке на скачивание;
2. Извлечь содержимое архива;
### 3. Docker-контейнер
1. Скачать и установить Docker при его отсутствии;
2. Запустить Docker Desktop;
3. Открыть терминал;
4. Для установки образа и контейнера основного сервиса перейти в директорию discovery_events-main\events-service и в ней ввести "docker compose up";
5. А для сервиса статистики перейти в директорию discovery_events-main\stats-service-parent\stats-service и также ввести "docker compose up".

> [!NOTE]
> Проверьте, что все зависимости и сторонние библиотеки, указанные в pom.xml загружены на ваш ПК;
> 
> Убедитесь, что у вас установлена Java 17, необходимая для запуска программы.
